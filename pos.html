Pos System

Shown above is a POS made with the Java GUI enviornment. It allows users to select a .txt menu file (in the proper format) and load that into a POS system. From there, the user is allowed to create, clear, and place orders by selecting items from the menu. Order totals are updated in real-time and credit cards are processed if they follow a check for validity. The code uses efficient object-oriented design principles as well as a MVC enviornment. Once a user creates an order and a valid [mock] credit card is used, a confirmation reciept is printed.
